Plugin administrativo para WordPress com interface React proprietaria DRA.

== Description ==
Painel administrativo em React integrado ao menu do WordPress para configuracoes internas.

== Installation ==
1. Envie a pasta do plugin para /wp-content/plugins/.
2. Ative o plugin no menu Plugins.
3. Acesse Painel DRA no admin.
