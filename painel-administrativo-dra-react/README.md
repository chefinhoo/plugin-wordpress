# Painel Administrativo DRA (React)

Plugin separado em React baseado no `panel-fix.zip`, com telas para notícias, categorias, novo post, módulos e configurações.

## Inclui
- Bootstrap PHP do plugin
- App React para o admin
- CSS original do painel com referências externas removidas
- Comportamento administrativo portátil
- REST nativo do WordPress para posts, categorias e opções

## Estrutura
- `painel-administrativo-dra-react.php`
- `includes/class-dra-react-plugin.php`
- `includes/class-dra-react-rest.php`
- `assets/dist/admin-original.css`
- `assets/dist/admin-behavior.js`
- `assets/dist/admin-react-app.js`

## Instalação
1. Copie a pasta do plugin para `wp-content/plugins/`.
2. Ative no painel do WordPress.
3. Acesse o menu Painel DRA (React).
